Como funciona

Simples do início ao fim

Você cria uma sessão, compartilha o link e os registros chegam em tempo real. Tudo centralizado em um dashboard limpo.

1
Crie uma sessão de triagem

No dashboard, abra uma nova sessão. O sistema gera automaticamente um link único e um QR Code para ela.

2
Compartilhe o link ou QR Code

Distribua para quem precisa preencher o formulário. Acesso direto pelo celular, sem instalação.

3
Acompanhe os registros em tempo real

Cada submissão aparece no seu dashboard imediatamente, com fotos, dados e metadados organizados.

4
Encerre e arquive com segurança

Quando a triagem termina, encerre a sessão. Apenas dados selecionados* ficam armazenados e auditáveis.
